# Embedded Document Sampling Notes (2025-11-12)

각 카테고리에서 대표 문서 1편씩 1-2페이지를 확인하여 핵심 키워드를 정리했습니다.

## OLED
- 문서: `OLED_2207.02479v1_2022-07-06.pdf`
- 핵심 주제: OLED 기반 양자 센싱, 스핀 기반 자기장 이미징, 고해상도 센서 구현
- 키워드: quantum sensing, magnetometry, spin-based imaging, organic light emitting diode
- 테스트 아이디어: "OLED로 자기장을 측정하는 연구를 요약해줘", "OLED를 활용한 양자 센서의 장점은?"

## QLED
- 문서: `QLED_2305.05825v1_2023-05-10.pdf`
- 핵심 주제: 콜로이드 양자점 구조 엔지니어링, 고효율/안정성/드룹 제어 QLED
- 키워드: quantum dots, droop-free, aerobic-stable, structural engineering, printed electronics
- 테스트 아이디어: "QLED 드룹 현상을 줄이기 위한 구조적 접근을 설명해줘", "QLED 안정성을 높이는 재료 전략은?"

## MicroLED
- 문서: `MicroLED_2302.05801v1_2023-02-11.pdf`
- 핵심 주제: BLAST 웨이퍼 스케일 전사 공정, 광전자 이종 집적, 마이크로디스플레이 제조
- 키워드: wafer-scale transfer, heterogeneous integration, microdisplay, OWiC Technologies, Cornell
- 테스트 아이디어: "BLAST 공정이 microLED 제조에 주는 이점은?", "microLED 이종 집적 과정의 주요 단계는?"

## Perovskite
- 문서: `Perovskite_2311.09099v1_2023-11-15.pdf`
- 핵심 주제: 메조 구조 박막 페로브스카이트 태양전지 성능 분석, 실험+시뮬레이션 결합
- 키워드: thin-film, device simulation, performance analysis, experimental validation
- 테스트 아이디어: "페로브스카이트 태양전지 성능을 개선한 실험-시뮬레이션 접근법은?", "메조 구조 페로브스카이트의 장점은?"

## Graphene
- 문서: `Graphene_2006.11563v1_2020-05-03.pdf`
- 핵심 주제: 그래핀 두께와 안테나 EM 모델링, THz 대역 시뮬레이션, 2D vs 3D 모델 비교
- 키워드: graphene thickness, THz antenna, electromagnetic modelling, simulation methodologies
- 테스트 아이디어: "그래핀 안테나 설계에서 두께가 미치는 영향은?", "그래핀 기반 THz 안테나 시뮬레이션 방법을 설명해줘"

## Display

- 키워드: stereoscopic dataset, HDR, immersive display, AR/VR benchmarking
- 테스트 아이디어: "스테레오 표현용 멀티 익스포저 데이터베이스의 특징은?", "VR/AR 디스플레이 평가를 위한 공개 데이터셋은?"
