# RAG Functional Test Question Set (2025-11-12)

문서 인벤토리와 샘플 내용을 기반으로, 실제 RAG 동작 테스트에 사용할 질문 세트를 정리했습니다.
각 질문은 **주요 사용 시나리오**에 맞추어 설계되었으며, 평가 시 확인해야 할 포인트를 함께 제시합니다.

---

## 1. 특정 논문 기반 질문 (Single-Paper Focus)
| 질문 | 예상 문서 | 평가 포인트 |
|------|-----------|-------------|
| OLED를 이용한 양자 자기장 이미징 연구를 요약해줘. | `OLED_2207.02479v1_2022-07-06.pdf` | 해당 논문 중심 요약, 양자 센싱/스핀 기반 장점 언급, 정확한 citation |
| QLED에서 드룹(droop) 현상을 줄이기 위한 구조적 접근을 설명해줘. | `QLED_2305.05825v1_2023-05-10.pdf` | 콜로이드 양자점 구조 엔지니어링, 안정성/드룹 제어 전략 포함 |
| BLAST 공정이 microLED 제조에 주는 이점을 정리해줘. | `MicroLED_2302.05801v1_2023-02-11.pdf` | 웨이퍼 스케일 전사, 이종 집적 절차 설명 |
| 메조 구조 페로브스카이트 태양전지의 성능 개선 방법은? | `Perovskite_2311.09099v1_2023-11-15.pdf` | 실험+시뮬레이션 접근법, 성능 지표, 핵심 파라미터 |
| 그래핀 두께가 THz 안테나 모델링에 미치는 영향은? | `Graphene_2006.11563v1_2020-05-03.pdf` | 2D vs 3D 모델 비교, THz 적용 사례 |
| VR/AR 디스플레이용 스테레오 데이터베이스의 특징을 알려줘. | `Display_2206.11095v1_2022-06-22.pdf` | 데이터셋 구성, HDR/멀티뷰 장점, 활용 시나리오 |

**체크리스트**
- 단일 문서 중심 요약인지 확인
- 문서 내 핵심 용어/수치/저자 정보 포함 여부
- Citation이 정확한 페이지를 가리키는지 검증

## 2. 주제/키워드 기반 논문 검색 (Topic-Level Retrieval)
| 질문 | 대상 범위 | 기대 결과 |
|------|-----------|-------------|
| MicroLED 웨이퍼 스케일 전사 기술의 최신 동향은? | MicroLED 전 범위 | 최근(2023~2025) 문서 위주, BLAST 등 공정 비교 |
| 그래핀 기반 THz 안테나 설계 방법을 정리해줘. | Graphene 전 범위 | 다양한 그래핀 안테나 구조, 두께/전도도 영향 |
| OLED 효율 향상을 위한 대표 전략을 알려줘. | OLED 전 범위 | 재료/구조/공정별 전략, 최소 2개 이상 문서 |
| Perovskite 태양전지에서 시뮬레이션 활용 사례를 알려줘. | Perovskite 전 범위 | 실험+시뮬레이션 결합 사례, 모델링 기법 |
| QLED 안정성 확보를 위한 재료/구조 연구를 소개해줘. | QLED 전 범위 | 안정성, 산소 내성, 패시베이션 전략 |
| VR/AR용 고해상도 디스플레이 연구의 주요 이슈는? | Display 전 범위 | 멀티뷰/HDR/자연영상 데이터셋, 사용자 경험 |

**체크리스트**
- 여러 문서에서 근거를 가져오는지 (출처 다양성 ≥ 2.5 목표)
- Hybrid Search + Reranker 결과가 주제에 맞는지 확인
- 질문 분류기가 `normal` 또는 `complex`로 올바르게 분류했는지 검증

## 3. 다문서 인사이트/비교 질문 (Multi-Document Insight)
| 질문 | 기대 포인트 |
|------|-------------|
| OLED, QLED, MicroLED의 제조 공정 차이를 비교해줘. | 각 기술별 핵심 공정, 장단점, 최소 3개 이상 문서 사용 |
| MicroLED와 QLED의 신뢰성 향상 전략을 비교해줘. | 안정성/드룹/패키징 전략 비교, 문서별 근거 명시 |
| Perovskite와 OLED의 효율 향상 접근법을 대조해줘. | 에너지/조명 분야 차이, 실험 vs 시뮬레이션 전략 |
| VR/AR 디스플레이와 MicroLED 기술의 융합 사례를 설명해줘. | Display+MicroLED 문서 결합, 응용 사례 제시 |
| 그래핀 기반 THz 안테나와 MicroLED 통신 인터페이스 가능성을 논의해줘. | Graphene+MicroLED 문서 조합, 상호 보완성 평가 |

**체크리스트**
- 최소 3개 이상의 고유 문서를 활용했는지 확인
- 비교 항목이 균형 있게 다뤄졌는지 (장점/단점, 공통점/차이점)
- Citation이 비교 대상별로 분리되어 있는지 검증

## 4. Exhaustive Retrieval & 파일 리스트
| 질문 | 기대 포맷 |
|------|------------|
| 모든 OLED 관련 논문을 파일 리스트로 보여줘. | Markdown 테이블, 관련도/청크 수 |
| MicroLED 웨이퍼 전사 공정을 다루는 모든 논문 목록을 알려줘. | 파일명+관련도, 최소 10개 |
| Perovskite 태양전지 시뮬레이션 연구 전체를 정리해줘. | 시뮬레이션 포함 문서 우선 정렬 |
| QLED 드룹 현상 연구 전체를 리스트로 반환해줘. | Droop 키워드 포함 문서 중심 |

**체크리스트**
- `_is_exhaustive_query`가 트리거되어 파일 리스트를 반환하는지
- File Aggregation 결과의 점수 및 청크 수가 합리적인지
- `enable_file_aggregation=true` 설정이 실제로 적용되는지 확인

## 5. 엣지/검증용 질문
| 질문 | 목적 |
|------|------|
| 그래핀이나 OLED가 아닌 전혀 다른 주제를 물으면 어떻게 응답하나요? (예: "양자컴퓨터 최신 연구") | 비존재 주제 처리, 안전 응답 |
| 특정 연도/버전 필터 질문 (예: "2024년 이후 MicroLED 논문") | 메타데이터 활용, 날짜 인식 여부 |
| 영어/한글 혼합 질문 (예: "Explain the droop mitigation strategies for QLEDs in Korean") | 다국어 처리 및 출력 언어 확인 |
| 문서에 없는 수치 질문 (예: "이 논문의 EQE 50% 달성 조건은?") | Hallucination 방지, 재질문/에러 처리 |

## 잠재적 누락/주의사항
- 현재 코퍼스는 PDF만 포함되어 있으며, PPTX/XLSX/TXT 자료는 임베딩되지 않음 → 문서 타입별 기능 테스트 시 주의 필요
- 메타데이터(저자, 기관, 연도) 검색 기능은 아직 구현되지 않았으므로 질문 설계 시 문서명/내용 기반으로 제한
- Graphene/Display 카테고리는 OLED/LED와 직접 연계되지 않을 수 있으므로, 다문서 질문 설계 시 컨텍스트 연결을 명확히 해야 함
- 최신(2025) 문서가 다수 포함되어 있으므로 LLM이 최신 용어/동향을 올바르게 요약하는지 확인
